# Professional Design System for Legal Case Management\n\nA comprehensive \"Google Docs for litigation\" design system built with Material-UI, featuring professional aesthetics optimized for legal document work.\n\n## Overview\n\nThis design system provides a cohesive set of components, tokens, and patterns specifically designed for legal case management software. It combines the clean, focused aesthetic of Google Docs with professional micro-interactions and accessibility features.\n\n## Key Features\n\n- **Google Docs-Inspired Aesthetics**: Clean, professional interface optimized for legal document work\n- **Comprehensive Theme System**: Light/dark mode with system preference detection\n- **Professional Micro-interactions**: Subtle animations and hover effects (120-160ms timing)\n- **Accessibility-First**: WCAG 2.1 AA compliant with proper contrast ratios\n- **Legal-Focused Colors**: Status colors for cases, priority levels, and document states\n- **Professional Typography**: Inter font stack optimized for legal document readability\n- **Consistent Spacing**: 8px base grid system for perfect alignment\n\n## Design Tokens\n\n###
